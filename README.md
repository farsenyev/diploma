# Accessibility Checker

Демонстрационный веб-сервис для автоматизированной проверки доступности веб-страниц.

Приложение анализирует HTML и runtime-состояние страницы и выявляет распространённые проблемы доступности интерфейсов (Accessibility / a11y).

---

# Основные возможности

Сервис выполняет проверку следующих аспектов доступности:
1. Альтернативный текст изображений
2. Структура страницы
3. Цветовая контрастность
4. Клавиатурная навигация
5. Доступность мультимедиа
6. Масштабируемость интерфейса

---

# Архитектура проекта

Проект разделён на два независимых приложения:


### Backend

Backend используется только для получения **runtime snapshot страницы**.

Он:

1. открывает страницу через Puppeteer
2. собирает данные DOM
3. собирает вычисленные стили
4. возвращает snapshot на frontend

Проверки выполняются на стороне клиента.

### Frontend

Frontend:

1. отправляет URL на backend
2. получает snapshot страницы
3. запускает модули проверки
4. отображает результаты анализа

# Установка и запуск

## 1. Клонирование репозитория

```
git clone <repo-url>
cd project
```
 ## 2. Backend
 ```
 cd backend
 pnpm i
 pnpm run start
 ```
Backend будет доступен по адресу
```http://localhost:3001```
 ## 3. Frontend
``` 
cd diploma
pnpm i
pnpm run dev
```
Frontend будет доступен по адресу
```http://localhost:5173```
